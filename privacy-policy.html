YouTube Parts Finder

Privacy Policy — Last updated: June 2026

YouTube Parts Finder is a free Chrome extension that automatically finds previous and next parts of YouTube Shorts series.

Data We Collect

We collect no personal data whatsoever.

The extension does not collect, store, transmit, or share any personal information about you.

Your API Key

Your YouTube Data API key is stored exclusively on your own device using Chrome's built-in secure storage. It is never transmitted to any server operated by us and is never shared with any third party.

YouTube API Usage

The extension makes calls directly from your browser to the YouTube Data API v3 using your own API key. These calls are subject to Google's own privacy policy at policies.google.com/privacy.

Browsing Data

The extension reads the current YouTube page URL only to detect whether you are watching a Shorts video. This data never leaves your browser.

No Tracking

We do not use any analytics, tracking pixels, cookies, or third party services of any kind.

Contact

If you have any questions about this privacy policy please open an issue on our GitHub repository.
